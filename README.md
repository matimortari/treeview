# TreeView

A simple CLI utility written in Go that displays a directory tree structure, excluding common development folders.

## Features

- Single standalone binary (no dependencies)
- Recursively prints directory structure
- Directories listed before files
- Alphabetically sorted
- Ignores dot-prefixed hidden files and folders
- Excludes common clutter folders:
  - node_modules
  - .git
  - .nuxt
  - .next
  - dist
  - build
  - bin
  - obj
  - .venv
  - out
  - .output
